Help

  • ❶ Press START button to start the game.
  • ❷ One time started the game: Press START button to restart the game.
  • ❸ One time started the game: Hold START button to finish the game.
  • ❹ At any time: Press STRICT button to enable/disable the strict mode.
  • ❺ At any time: Press SKILL LEVEL switch to change the game difficulty.
